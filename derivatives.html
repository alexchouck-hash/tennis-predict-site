Markets: Match winner · Derivatives (set 1 · exact score · totals)

Fair values are computed in closed form from the same engine that prices the match board — set-1 winner, exact set score, and expected games per match. Strategy column: BUY at ask only when the fair value clears the ask plus fees with margin (rare on tight books, by design); rest bid = a maker limit order below fair value on soft, wide-spread books (exact-score books are typically 5-8¢ wide — crossing the spread there gives the edge away). Total-games fair values are shown for reference and not traded pending calibration. Paper/analytical only — not financial advice.
